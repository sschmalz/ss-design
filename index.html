ABOUT ME


Hello, I'm web designer and front-end developer in Portland, Oregon, focused on creating beautiful responsive sites. My background spans from painting and illustration to logo design to ebook development to web design. I studied art and writing in college, and have worked in web design and marketing since graduating.

When I'm not working on websites, you can find me knitting, reading young adult novels, working in my veggie garden, or going on adventures with my dog.

SKILLS

HTML5 / CSS3

I spend my days writing HTML & CSS, playing with divs and styles. I'm completely comfortable laying out sites in divs within divs, with some more divs; but please don't make me develop a table-based site.

Responsive

I won't pretend to be an expert in responsive design; everything's constantly changing, but I try to keep up and create sites that will not completely explode when the next iPhone with crazy new dimensions comes out.

Design & Development

I started out as a web designer with just enough coding knowledge to do damage; but now I've moved more into the development side, and am just as comfortable creating Photoshop mockups as I am coding out those mockups.

Bootstrap

Bootstrap is my go-to framework. I'm extremely comfortable designing and developing responsive sites with it. Even my custom Wordpress themes are built on Bootstrap.

Foundation

Foundation is growing on me (I've even designed this very site with it), but I'm not quite as comfortable with it, as Bootstrap, but we're getting there. The wealth of plug-ins, add-ons, and templates makes it a pretty awsome framework to start a site with.

Wordpress

I've set up Wordpress sites for clients using themes developed by others, as well as developed custom themes for clients based on Roots Bootstrap framework.

I also have basic PHP knowledge (enough to create custom Wordpress themes), fairly extensive Adobe ColdFusion development experience, ebook creation and coding knowledge, and a pretty strong writing background.
